# 전기기사 암기·정리 노트

> 이 파일에 앞으로 정리하는 내용을 모두 누적합니다.

---

## 설비불평형률 (Facility Imbalance Rate)

> **읽는 순서:** 0단계(기초) → 1단계(규칙) → 2단계(예외 4개) → 3단계(암기법) → 4단계(계산)

---

### 0단계 — 전기를 모르는 사람도 이해하기 (초등학생 버전)

#### 전기는 왜 3줄(3상)로 오나?

집이나 공장에 오는 큰 전기는 보통 **전선 3개**로 들어옵니다.  
이걸 **3상 전기**라고 부릅니다.

쉽게 비유하면:

```
        A선  ───────  친구 A가 들고 가는 가방
        B선  ───────  친구 B가 들고 가는 가방
        C선  ───────  친구 C가 들고 가는 가방
```

세 친구가 **같은 무게**를 나눠 들면 편합니다.  
한 친구만 무겁게 들면 그 친구만 힘들어지죠.  
전기도 똑같습니다. **세 줄이 고르게** 쓰이면 안전하고 효율적입니다.

---

#### 「불평형」이란? = 한쪽만 너무 무거운 것

| 상태 | 비유 | 전기에서는 |
|------|------|------------|
| **평형** | A, B, C가 비슷한 무게 | 세 상이 비슷하게 전기 사용 |
| **불평형** | A만 10kg, B·C는 2kg | 한 상만 전기를 너무 많이 씀 |

불평형이 크면:
- 전선·변압기가 **한쪽만 과열됨**
- 다른 집·공장 전기에도 **영향**을 줄 수 있음  
→ 그래서 나라에서 **“너무 한쪽으로 치우치지 마!”** 라고 규칙을 만든 것이 **설비불평형률**입니다.

---

#### kVA가 뭐야? (아주 간단히)

- **k** = 1000 (킬로)
- **V** = 볼트 (전압)
- **A** = 암페어 (전류)

**kVA**는 “전기를 **얼마나 많이 쓰는지**”를 나타내는 **용량 단위**입니다.  
시험에서는 **숫자만 비교**하면 됩니다.

```
100 kVA 이하  =  작은 전기 사용
100 kVA보다 큼 =  큰 전기 사용
```

---

#### 저압 · 고압 · 특고압 — 전기가 오는 “길”의 크기

| 이름 | 아주 쉬운 비유 | 한 줄 설명 |
|------|----------------|------------|
| **저압** | 동네 작은 길 | 일반 건물·가정에 많이 씀 (전압 낮음) |
| **고압** | 시내 큰 도로 | 공장·중형 건물 (전압 더 높음) |
| **특고압** | 고속도로 | 대형 공장·산업단지 (전압 가장 높음) |

전압이 높을수록 **많은 전기를 멀리 보낼 수 있지만**,  
불평형이 생기면 **더 큰 문제**가 될 수 있어서 규칙이 더 세밀합니다.

---

#### 단상부하 vs 3상부하

| 종류 | 비유 | 예시 |
|------|------|------|
| **단상부하** | 가방을 **한 친구한테만** 실어 줌 | 히터, 조명, 콘센트(일반) |
| **3상부하** | 가방을 **세 친구가 나눠** 듦 | 큰 모터, 3상 전동기 |

**설비불평형률**은 주로 **단상부하가 얼마나 한쪽으로 치우쳤는지**를 봅니다.

---

### 1단계 — 기본 규칙 (문제 1)

| 항목 | 내용 |
|------|------|
| 적용 대상 | 3상 3선식 · 3상 4선식 |
| 기준 한도 | **30% 이하** |

**한 줄 요약:** 3상 전기 쓰는 곳은 **불평형이 30%를 넘으면 안 된다** (단, 아래 4가지 예외 제외).

---

#### 불평형률 공식 — 단계별로 풀기

```
불평형률 [%] = (가장 무거운 상 − 가장 가벼운 상) ÷ 평균 × 100
```

**1단계:** A, B, C 각 상에 얼마나 전기가 걸렸는지 구한다.  
**2단계:** 그중 **제일 큰 값**과 **제일 작은 값**을 찾는다.  
**3단계:** (큰 값 − 작은 값)을 구한다.  
**4단계:** 세 상의 **평균**으로 나눈다.  
**5단계:** 100을 곱하면 **%**가 나온다.

**초등 비유로 계산**

```
A상: 8kg,  B상: 5kg,  C상: 4kg

차이 = 8 − 4 = 4kg
평균 = (8 + 5 + 4) ÷ 3 = 5.67kg

불평형률 = 4 ÷ 5.67 × 100 ≈ 70%
```

→ 30%보다 훨씬 크니까 **규칙 위반** (예외에 해당하지 않으면).

---

### 2단계 — 30% 규정을 **안 지켜도 되는** 4가지 예외 (문제 2)

> **핵심:** 대부분은 30%를 지켜야 하지만, **아래 4가지 상황**은  
> “괜찮아, 이 정도는 예외로 봐줄게” 하는 **면제 카드** 4장입니다.

#### 한눈에 보기

| # | 전압 | 핵심 조건 | 초등 비유 |
|---|------|-----------|-----------|
| ① | **저압** | 전용 변압기로 수전 | 우리 집만 쓰는 **개인 화장실** |
| ② | **고압·특고압** | 단상부하 **100 kVA 이하** | **작은 가방** 하나뿐 |
| ③ | **고압·특고압** | 최대−최소 **≤ 100 kVA** | 가방 무게 **차이가 작음** |
| ④ | **특고압** | 100 kVA 이하 변압기 2대 **역V결선** | **특별한 2개 도구** 결선 |

---

#### ① 저압 수전에서 전용 변압기 등으로 수전하는 경우

##### 📖 법 조문 뜻 (시험 답)

저압으로 전기를 받을 때, **그 건물·공장만 쓰는 전용 변압기**로 수전하면  
설비불평형률 30% 규정을 **적용하지 않아도 된다**.

##### 🧒 초등학생 비유

아파트에 전기가 오는 방법이 두 가지라고 생각해 보세요.

```
[방법 A] 옆집과 전기를 같이 쓰는 큰 변압기
         → 내가 전기를 많이 쓰면 옆집에도 영향

[방법 B] 우리 집 전용 변압기 (전용 = 우리만 씀)
         → 우리 집 안에서만 조절, 밖에 덜 피해
```

**전용 변압기** = **“우리 공장·건물만을 위한 전기 변환기”**

##### 🔍 왜 예외인가?

1. 전기를 **우리 건물 안에서** 먼저 받아서 씀  
2. 불평형이 생겨도 **다른 사람 전기선에 바로 안 닿음**  
3. 그래서 국가도 “저압 + 전용변압기면 30% 규칙 안 봐도 됨”이라고 함

##### ✅ 시험에서 체크할 것

- **저압**인가? ✓  
- **전용** 변압기인가? ✓  
- 둘 다 맞으면 → **예외 ①**

##### ❌ 헷갈리지 말 것

- 그냥 저압만으로는 예외 **아님** → **전용 변압기**가 꼭 있어야 함  
- 100 kVA 숫자와 **무관** (①에는 100이 없음!)

---

#### ② 고압 및 특고압 수전에서 100 kVA 이하의 단상 부하인 경우

##### 📖 법 조문 뜻

고압·특고압으로 전기를 받는데, **단상부하 전체가 100 kVA 이하**이면  
불평형률 30%를 넘어도 **괜찮다**.

##### 🧒 초등학생 비유

고속도로(고압·특고압)에 작은 자전거(단상부하) 한 대만 다니는 것과 같아요.

```
고속도로 = 전기가 크게 흐르는 길
자전거   = 100 kVA 이하의 작은 단상부하

→ 자전거 하나 정도는 고속도로에 큰 방해가 안 됨
```

##### 🔍 왜 예외인가?

- **절대적으로 작은** 단상부하는  
  아무리 불평형이 있어도 **전체 전기 계통에 영향이 거의 없음**  
- “애초에 너무 작아서 신경 안 써도 됨”

##### ✅ 시험에서 체크할 것

- **고압 또는 특고압**인가? ✓  
- **단상부하**인가? ✓  
- 단상부하 용량이 **100 kVA 이하**인가? ✓  

##### 📊 숫자 예시

| 단상부하 | ② 예외? |
|----------|---------|
| 50 kVA | ✅ 예외 |
| 100 kVA | ✅ 예외 (이하 포함) |
| 101 kVA | ❌ 예외 아님 |

---

#### ③ 특고압 및 고압 수전에서 단상부하 용량의 최대와 최소의 차가 100 kVA 이하인 경우

##### 📖 법 조문 뜻

고압·특고압 수전에서, A·B·C 상 중  
**가장 큰 상 − 가장 작은 상 ≤ 100 kVA**이면 예외.

##### 🧒 초등학생 비유 — ②와 가장 헷갈리는 부분!

세 친구가 가방을 드는 상황:

```
[②의 질문]  "가방 전체가 작아?"  →  가방 자체가 100 이하
[③의 질문]  "친구들 차이가 작아?" →  제일 무거운 − 제일 가벼운 ≤ 100
```

**②** = **총량**이 작다  
**③** = **격차**가 작다 (각각은 커도 됨!)

##### 🍎 사과로 이해하기

| A상 | B상 | C상 | 최대−최소 | ③ 예외? | 설명 |
|-----|-----|-----|-----------|---------|------|
| 200 | 180 | 150 | 50 | ✅ | 차이 50 → 100 이하 |
| 300 | 280 | 220 | 80 | ✅ | 차이 80 → OK |
| 300 | 250 | 100 | 200 | ❌ | 차이 200 → 너무 큼 |
| 80 | 70 | 60 | 20 | ✅ (③) | ②도 ✅ (전체 작음) |

##### 🔍 왜 예외인가?

- 각 상이 200, 300처럼 **커도**  
  상호 간 **차이만 100 이하면** “어느 정도 균형”이라고 봄  
- ②는 “아예 작은 집”, ③은 “큰 집이지만 **고르게** 쓰는 집”

##### ✅ 시험에서 체크할 것

1. 고압 또는 특고압?  
2. A, B, C 상 부하 각각 구하기  
3. **(최대 − 최소) ≤ 100** 인지 확인  

---

#### ④ 특고압 수전에서는 100 kVA 이하의 단상 변압기 2대로 역 V결선하는 경우

##### 📖 법 조문 뜻

**특고압** 수전에서 **100 kVA 이하 단상변압기 2대**를  
**역 V결선(오픈 델타)** 으로 연결하면 예외.

##### 🧒 초등학생 비유

보통 3상 전기는 **변압기 3대**가 제일 좋습니다 (세 친구가 각자 한 개씩).

```
정상(3대):  [변압기A] [변압기B] [변압기C]  →  고르고 좋음

역V(2대):  [변압기1] [변압기2]  →  3상을 억지로 만듦
                                    →  원래 불균형하기 쉬움
```

**역 V결선** = 변압기 **2개만**으로 3상 전기를 만드는 **특수한 꼼수 결선**

비유: 세 명이 해야 할 일을 **두 명이** 하는 것 →  
당연히 한쪽이 더 힘들어지기 쉽습니다 (불평형).

##### 🔍 왜 예외인가?

- 역V결선은 **값이 싸고** 소규모 공장에서 자주 씀  
- 하지만 구조상 불평형이 **원래 생기기 쉬움**  
- 그래서 **“특고압 + 100 kVA 이하 + 2대”** 처럼  
  **작을 때만** 예외로 허용

##### ⚠️ ④만의 특별 조건 (시험 함정)

| 조건 | ④에 필요? |
|------|-----------|
| **특고압** | ✅ 필수 (고압만으론 ❌) |
| 변압기 **2대** | ✅ |
| **역 V결선** | ✅ |
| 각 변압기 **100 kVA 이하** | ✅ |

##### 🖼️ 역 V결선 그림 (개념)

```
        고압 전원 (3상)
           │ │ │
           ▼ ▼
      ┌────┴─┴────┐
      │  변압기 1  │
      │  변압기 2  │  ← 2대만 연결 (3번째는 없음)
      └─────┬─────┘
            ▼
        저압 3상 출력
```

### 3단계 — 암기법

#### 🎯 메인 암기문장: **「저전 고단 차백 역V」**

입으로 크게 읽어 보세요:  
**“저-전 / 고-단 / 차-백 / 역-V”**

| 글자 | 풀어쓰기 | 해당 | 초등 한 줄 |
|------|----------|------|------------|
| **저**전 | **저**압 + **전**용 변압기 | ① | 우리만 쓰는 변압기 |
| **고**단 | **고**압·특고압 + **단**상 100↓ | ② | 작은 단상부하 |
| **차**백 | **차**이 **백**(100) 이하 | ③ | 상 간 차이 100↓ |
| **역**V | **역** V결선 (특고압, 2대) | ④ | 2개 변압기 특수결선 |

---

#### 🎵 리듬 암기 (외우기 쉬운 말)

```
저압은 전용,
고압은 단상 백,
차이도 백,
특고는 역V!
```

---

#### 🔢 숫자 암기: **100은 세 번**

```
②  단상부하     ≤ 100
③  최대 − 최소  ≤ 100
④  변압기 1대당  ≤ 100  (2대 역V)
```

**①만 100이 없다** → “저전”은 숫자 없이 **전용**만 기억!

---

#### 🗺️ 전압 지도

```
저압     ──── ① 전용변압기
  │
고압     ──── ②③
  │
특고압   ──── ②③④  ← ④는 여기만!
```

**시험 함정:** ④를 고압에 쓰면? → **틀림!** 특고압만!

---

#### ⚔️ ② vs ③ — 반드시 구분

| | ② | ③ |
|---|----|----|
| **묻는 것** | 전체가 작나? | 차이가 작나? |
| **비유** | 가방이 작다 | 무게 차이가 작다 |
| **조건** | 단상 ≤ 100 | 최대−최소 ≤ 100 |
| **전압** | 고압·특고압 | 고압·특고압 |

**외우는 문장:**  
> “**②는 작은 집, ③은 고른 집**”

---

#### 🃏 4장의 면제 카드 — 스토리로 한 번에

```
🏠 ① 저전   : "우리 집 전용 변압기 있어요"     → 면제!
📦 ② 고단   : "단상부하가 작아요 (100↓)"       → 면제!
⚖️ ③ 차백   : "상 차이가 100 이하여요"         → 면제!
🔧 ④ 역V    : "특고압에서 작은 역V 2대예요"    → 면제!
```

4장 중 **하나라도** 해당되면 30% 규칙 **안 봐도 됨**.

### 4단계 — 계산 예시 (문제 3)

#### 회로 그림 (글로 표현)

```
        A ───────── B
       /│\         /│\
      / │ \       / │ \
     H  H  H     H  H  H  ... (히터들)
      \ │ /       \ │ /
       \│/         \│/
        C ───────── (모터 M: 3상)
```

| 선간 | 부하 |
|------|------|
| A−B | 히터 2.5 + 2.5 kVA |
| B−C | 히터 2 + 1.5 + 5.5 + 1.7 kVA |
| A−C | 히터 3.5 kVA |
| 3상 | 모터 5.7 kVA |

#### 계산 (모범답안 따라가기)

```
분자 (차이) = (2.5+2.5+3.5) − (2+1.5+1.7) = 8.5 − 5.2 = 3.3

분모 (평균) = 1/3 × (8.5 + 5.2 + 5.5 + 5.7) = ...

불평형률 ≈ 39.76%
```

**39.76% > 30%** → 기본 규칙으로는 **안 됨**  
→ ①②③④ 예외에 해당하는지 확인  
→ 해당 없으면 **기준 초과 (불합격)**
