문제 050

저압 수전의 단상 3선식에서 중성선과 각 전압측 전선 간의 부하는 불평형 부하를 제한할 때 몇 [%]를 초과하지 않아야 하는가?

답: 40[%]

📌 덧붙인 설명 이 40%는 설비불평형률 30%와 다른 규정입니다.
• 30% = 3상 3선·4선식 전체의 설비불평형률 한도
• 40% = 저압 단상 3선식에서 중성선(N)과 한 상(전압측) 사이에 걸린 불평형 부하 한도
시험에서 숫자만 헷갈리면 틀립니다. "3상 전체 30%" vs "단상3선 N-상간 40%"로 구분하세요.
문제 051-1

저압·고압·특고압 수전의 3상 3선식 또는 3상 4선식에서 설비불평형률 한도는?

답: 30[%] 이하 (원칙)

📌 덧붙인 설명 단상 접속 부하 기준으로 각 상(A,B,C)에 걸린 부하를 계산한 뒤, (최대−최소)÷평균×100으로 구합니다. 30%를 넘으면 원칙적으로 불합격이지만, 아래 4가지 예외에 해당하면 30%를 넘어도 됩니다.

① 전기는 왜 3줄(3상)로 오나?

집·공장에 오는 큰 전기는 보통 전선 3개(A, B, C). 이걸 3상 전기라고 합니다.

        A선  ───────  친구 A가 들고 가는 가방
        B선  ───────  친구 B가 들고 가는 가방
        C선  ───────  친구 C가 들고 가는 가방
📌 덧붙인 설명 3상은 전력을 효율적으로 보내기 위한 방식입니다. 모터·대형 설비는 3상으로 돌아가고, 가정·조명·콘센트 같은 단상부하는 A, B, C 중 한 상에만 붙는 경우가 많습니다. 그래서 "한쪽 상만 무겁게 쓰는" 불평형이 생기고, 이를 제한하는 게 설비불평형률입니다.

② 「불평형」이란?

상태비유전기에서는
평형A,B,C 비슷한 무게세 상이 비슷하게 사용
불평형A만 10kg, B·C는 2kg한 상만 과다 사용
📌 덧붙인 설명 — 왜 규제하나? 한 상에만 부하가 몰리면 → 그 상의 전선·변압기 과열 → 화재·고장 위험.
중성선(N) 전류도 커져서 3상 4선식에서는 중성선 과열도 발생합니다. 그래서 "30% 이하로 맞춰라"는 국가 규정이 있습니다.

③ kVA가 뭐야?

  • k = 1,000   V = 볼트   A = 암페어
  • kVA = 전기 설비의 용량(크기) 단위. 시험에서는 숫자만 비교하면 됨.
100 kVA 이하  =  작은 부하
100 kVA보다 큼 =  큰 부하
📌 덧붙인 설명 예외 ②③④에 나오는 100은 모두 kVA(kW) 단위입니다. "100이하", "차이 100이하", "변압기 1대당 100이하" — 숫자 100이 세 번 등장합니다. ① 예외에는 100이 없습니다. 이것도 시험 함정!

④ 저압 · 고압 · 특고압

이름비유설명
저압동네 작은 길가정·일반 건물 (약 220/380V)
고압시내 큰 도로중형 공장·건물 (수 kV~수십 kV)
특고압고속도로대형 산업단지 (22.9kV, 66kV 등)
📌 덧붙인 설명 — 예외와 전압 관계 ① 저압만 해당 / ②③ 고압·특고압 / ④ 특고압만 (고압에 ④ 쓰면 틀림!)

⑤ 단상부하 vs 3상부하

종류비유예시
단상부하한 친구만 가방히터, 조명, 일반 콘센트
3상부하세 친구가 나눠 듦3상 모터, 대형 설비
📌 덧붙인 설명 설비불평형률은 단상부하가 A,B,C 중 어디에 얼마나 치우쳤는지를 봅니다. 3상 모터는 세 상에 고르게 걸리므로 불평형 계산에서 상대적으로 영향이 다릅니다.

기본 규칙 — 30% 이하

항목내용
적용 대상3상 3선식 · 3상 4선식

계산 기준단상 접속 부하로 각 상 계산

불평형률 공식

불평형률 [%] = (가장 큰 상 − 가장 작은 상) ÷ 평균 × 100
  1. A, B, C 각 상의 단상부하 합산
  2. 최대값, 최소값 찾기
  3. 차이 = 최대 − 최소
  4. 평균 = (A+B+C) ÷ 3
  5. 차이 ÷ 평균 × 100 = %

계산 예 (초등 비유)

A상 8kg, B상 5kg, C상 4kg
차이 = 8−4 = 4
평균 = (8+5+4)÷3 = 5.67
불평형률 = 4÷5.67×100 ≈ 70%  →  30% 초과 → 원칙 위반
📌 덧붙인 설명 — 왜 30%인가? 30%는 전력 계통에서 변압기·선로가 안전하게 운전할 수 있는 불평형 허용치입니다. 넘으면 전기사업자가 수전을 거부하거나 시설 변경을 요구할 수 있습니다. 단, 예외 4가지에 해당하면 30%를 넘어도 합격입니다.

① 저압 + 전용 변압기

법 조문

저압 수전에서 전용 변압기로 수전하면 설비불평형률 30% 규정 미적용.

비유 
[공용] 옆집과 변압기 공유 → 내 불평형이 옆집에 영향
[전용] 우리 건물만 변압기 → 밖에 영향 적음 → 규제 완화
왜 예외?

전기를 건물 안에서 먼저 받아 쓰므로, 불평형이 외부 계통에 미치는 영향이 작습니다.

체크
  • 저압 ✓   전용 변압기 ✓ → 예외 ①
  • 저압만으로는 ❌   100 숫자 무관
📌 덧붙인 설명 "전용"이 핵심입니다. 아파트 공용 변압기로 저압 수전하면 예외 아님. 공장·건물 전용 변압기가 따로 있을 때만 해당.

비유

고속도로에 자전거 한 대 — 전체 교통에 영향 거의 없음.

단상부하② 예외?

④ 특고압 + 100↓ 변압기 2대 역 V결선

법 조문

계산 예시 (문제 3 유형)

        A ───────── B
       /│\         /│\
     히터들       히터들
       \│/         \│/
        C ─── (모터 M: 3상)
구분부하 [kVA]
A−B 선간 히터2.5 + 2.5 = 5.0
B−C 선간 히터2 + 1.5 + 5.5 + 1.7 = 10.7
A−C 선간 히터3.5
3상 모터5.7 (3상부하)

계산 과정

① 각 상에 걸린 단상부하 합산 (선간 히터를 상별로 배분)
② 최대 상 = 8.5,  최소 상 = 5.2  (모범답안 기준)
③ 분자(차이) = 8.5 − 5.2 = 3.3
④ 분모(평균) = ⅓ × (8.5 + 5.2 + 5.5 + 5.7) = ...
⑤ 불평형률 ≈ 39.76%
📌 덧붙인 설명 — 판정 순서 1. 불평형률 계산 → 39.76% > 30% → 기본 규칙 위반
2. 예외 ①②③④ 해당 여부 확인
3. 하나도 해당 안 되면 → 불합격(기준 초과)
4. 하나라도 해당되면 → 30% 넘어도 합격

대용량 단상전열 — 전기사업자 협의 시설 (문제 052)

특고압·고압에서 대용량 단상전기로 등으로 30% 지키기 어려울 때, 전기사업자와 협의하여 시설합니다.

052-1 단상부하 1개

2차 역 V 접속에 의할 것. 다만 300 kVA 초과 금지.

📌 덧붙인 설명 단상부하가 1개뿐일 때는 변압기 2대로 역V 결선해서 공급. 용량은 300 kVA 이하여야 함.
052-2 단상부하 2개

스코트(Scott) 접속에 의할 것.

단, 1개 용량이 200 kVA 이하이면 부득이한 경우 보통 변압기 2대로 별개 선간에 접속 가능.

📌 덧붙인 설명 스코트 결선 = 단상 변압기 2대로 3상→2상 변환하는 결선 (변압기 단원 145번 기출). 예외 ④의 역V와 다름 — 052는 "협의 시 시설 방법", ④는 "30% 예외 조건".
052-3 단상부하 3개 이상

가급적 선로전류가 평형이 되도록 각 선간에 부하를 접속할 것.

📌 덧붙인 설명 부하를 A-B, B-C, A-C 선간에 최대한 고르게 분산시켜 불평형을 줄이라는 뜻.
✓

시험 체크리스트

  • 3상 3선·4선 기본 한도 30%
  • 저압 단상3선 N-상간 불평형 40% (별도!)
  • 예외 4개: 저전 · 고단 · 차백 · 역V
  • 100은 ②③④에만 (①엔 없음)
  • ④ 역V는 특고압만
  • ② = 전체 작음 / ③ = 차이 작음
  • ① = 저압 + 전용변압기
  • 052 협의: 1개→역V(300↓), 2개→스코트, 3개↑→평형
